Metodología BEM

Es una metodología de trabajo dentro de CSS.

Son las siglas de Bloque, Elemento y Modificador (BEM) y es una forma de escribir los selectores CSS de una forma más rápida y sin tender a la repetición del código para poder acceder a un selector específico.

Es una forma de nombrar las clases.

Para su aplicación se usa siempre CLASS en lugar de ID.

Sintaxis
.bloque_elemento--modificador

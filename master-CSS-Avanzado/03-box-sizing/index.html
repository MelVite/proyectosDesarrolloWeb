Box Sizing en CSS

La propiedad box-sizing indica cómo se debe calcular las medidas de un elemento.

Por defecto en el modelo de caja, el ancho y alto asignado a un elemento es aplicado solo al contenido de la caja del elemento. Si el elemento tiene algún borde/border o relleno/padding, este es añadido al ancho y alto del tamaño de la caja content-box. Esto significa que cuando se define el ancho y alto, se tiene que ajustar el valor definido para permitir cualquier borde a relleno que se pueda añadir.

Sintaxis

La propiedad box-sizing es especificada como una sola palabra clave elegida con dos posibles valores:

  • content-box Es el valor por defecto, las propiedades width y height NO incluyen el borde, relleno o margen.

  • border-box Las propiedades de width y height incluyen el contenido, relleno y borde pero no incluyen margén.

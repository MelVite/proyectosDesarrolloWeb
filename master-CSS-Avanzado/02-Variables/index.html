Variables en CSS

¿Qué es una variable?

Son pequeños espacios en memoria de un programa donde se puede guardar información.

Para definir una variable se debe hacer dentro de la pseudo clase root y se utliza la siguiente sintaxis:

:root { --nombre-variable: #7d009c; }

En este ejemplo tenemos una variable para un color, se puede guardar cualquier cosa.

Para aplicarlo en CSS para el titulo h1 es de la siguiente forma:

h1 { color: var(--nombre-variable); }

¿Qué es la pseudoclase root?

Es una clase que selecciona al elemento raíz que representa TODO EL DOCUMENTO, es decir, todo el HTML. De esta manera se definen variables de estilo de manera base.
